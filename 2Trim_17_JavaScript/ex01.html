Caixas de diálogo em JS

São três tipos de caixa de diálogos

  • aletr> exibe uma mensagem de alerta ao usuário
  • confirm: exibe uma mensagem de confirmação em que o usuário pode escolher entre ok ou cancelar
  • prompt: o usuário pode entrar dados na caixa de diálogo

É importante destacar que o JavaScript pode ser incorporado ao documento HTML pela tag script. O script JS pode ser interno, ou seja, dentro do documento HTML ou externo no qual será um arquivo externo js. Ainda é possível colocar comandos JS associados a eventos como no exemplo a seguir.
