JavaScript

JavaScript é uma linguagem de script multiplataforma, orientada a objetos, usada para tornar páginas da web interativas (por exemplo, ter animações complexas, botões clicáveis, menus pop-up, etc.). Há também versões mais avançadas do lado do servidor do JavaScript, como o Node.js, que permite adicionar mais funcionalidade a um site do que baixar arquivos (como colaboração em tempo real entre vários computadores). Dentro de um ambiente host (por exemplo, um navegador da web), o JavaScript pode ser conectado aos objetos de seu ambiente para fornecer controle programático sobre eles.

O JavaScript contém uma biblioteca padrão de objetos, como Array, Map e Math, e um conjunto central de elementos de linguagem, como operadores, estruturas de controle e instruções. O JavaScript central pode ser estendido para uma variedade de propósitos, complementando-o com objetos adicionais; por exemplo:

  • O JavaScript do lado do cliente estende a linguagem central fornecendo objetos para controlar umnavegador e seu Document Object Model (DOM). Por exemplo, as extensões do lado do cliente permitem que umaplicativo coloque elementos em um formulário HTML e responda a eventos do usuário, como cliques domouse, entrada de formulário e navegação de página.
  • O JavaScript do lado do servidor estende a linguagem central fornecendo objetos relevantes para aexecução do JavaScript em um servidor. Por exemplo, as extensões do lado do servidor permitem que umaplicativo se comunique com um banco de dados, forneça continuidade de informações de uma invocaçãopara outra do aplicativo ou execute manipulações de arquivo em um servidor.

Isso significa que, no navegador, o JavaScript pode alterar a aparência da página da web (DOM). E, da mesma forma, o JavaScript do Node.js no servidor pode responder a solicitações personalizadas enviadas por código executado no navegador.

Caixas de diálogo em JS

São três tipos de caixa de diálogos

  • alert: exibe uma mensagem de alerta ao usuário
  • confirm: exibe uma mensagem de confirmação em que o usuário pode escolher entre ok ou cancelar
  • prompt: o usuário pode entrar dados na caixa de diálogo

É importante destacar que o JavaScript pode ser incorporado ao documento HTML pela tag script. O script JS pode ser interno, ou seja, dentro do documento HTML ou externo no qual será um arquivo externo js. Ainda é possível colocar comandos JS associados a eventos como no exemplo a seguir.

Variáveis

Primeiramente é recomendado declarar a variável antes de usá-la no JavaScript, mesmo que a linguagem seja capaz de declará-la automaticamente, existem 3 tipos de declarações manuais:

  • var - é um método mais antigo de declaração cujo é recomendada apenas ao ser destinada à navegadores antigos
  • let - é a versão do var mais recente, e produz quase o mesmo resultado
  • const - é utilzado para manter um valor constante, por exemplo definir um tempo padrão para todas as funções

A diferença entre var e let, além de um ser recomendado para navegadores antigos e outro para modernos, é que o var faz com que a variável esteja disponível em toda a função, enquanto o let somente no bloco onde foi declarado.
Ou seja, com o let você cria varíaveis locais enquanto o var faz com que ela funcione em toda a função.


Estruturas Condicionais

No JavaScript é possível criar estruturas condicionais, que, basicamente, são caminhos que o programa pode execurtar caso certos parâmetros sejam respondidos.

São estas as estruturas:

  • if - que faz executar certo bloco de comandos caso as condições sejam efetivadas
  • else - deve ser utilziado após o if, este faz executar o programa executar um outro certo bloco de comandos caso as condições do if não sejam efetivadas
  • else if - usado após um if, faz com que só seja verificado caso o primeiro if não execute.
  • switch - é uma estrutura complexa que possibilita ter acesso a diversos caminhos dependendo do valor dos parâmetros, por exemplo fazer uma estrutura condicional enorme só usando os dias da semana.

Laços de Repetição

Os Laços de Repetição no JavaScript servem para ficarem rodando um mesmo conjunto de blocos ou um único bloco até que a condição do laço seja completado.

Estes são os tipos de laços:

  • for - que faz executar um código por x vezes até que x alcance o valor definido, ademais, x pode ser usando como controlador, por exemplo, na manipulação de arrays.
  • while - mantém o código rodando até que sua condição invalide.
