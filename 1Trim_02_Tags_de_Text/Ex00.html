Desenvolvimento Web

Tecnologias utilizadas:

HTML: Linguagem de marcação de hipertexto.

CSS: Linguagem para a personalização de texto.

JavaScript: Linguagem para ações lógicas.

Front-End

Front-End corresponde a área de programação voltada à interface gráfica,
diretamente relacionado com ainteração com o usuário.

O desenvolvedor deste campo utiliza linguagens como HTML, CSS e JS.

Back-End

Back_End, como o próprio nome diz, é o código que está por trás("back"), parte responsável pela lógica do programa.

O programador Back-End trabalha com linguagens como java e python.

FullStack

FullStack é a junção das duas áreas, Front-End e Back-End, formando todo o programa.
Resumidamente, união da parte visual e lógica.

Desse modo, o programador FullStack pode criar um site completo.
