O que é tabela

A tag <table> serve para representar dados tabulares, ou seja, dados serão apresentadados em formato bidimensional
sendo composta por linhas e colunas de células.

Componentes

<table> - serve para iniciar a criação de uma tabela
      <caption> - serve para colocar um título à tabela
      <thead> - serve para iniciar a estrutura das colunas da tabela
            <tr> - serve para criar uma lista para a criação das colunas
                  <th scope="col"> - serve para criar uma coluna e entre as tags é colocado o nome
      <tbody> - serve para iniciar a estrutura das linhas da tabela
            <tr> - serve para iniciar a estrutura de uma linha da tabela
                  <th scope="row"> - serve para criar uma linha e entre as tags é colocado os dados da coluna
                  <td> - serve para escrever os dados da coluna relacionada a posição da tag (1º tag 1º coluna)
      <tfoot> - serve para iniciar a estrutura do rodapé
