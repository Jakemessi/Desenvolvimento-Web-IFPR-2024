Hyperlinks

São referências dentro de um documento em hipertexto para outras partes do mesmo documento ou outros documentos.

Os links podem ser:

  • Remotos
  • Internos
  • Locais
  • Endereços de correio eletrônico
  • Arquivos para download

Hyperlinks Remotos

São hyperlinks que ligam para outros documentos disponíveis na web. Exemplo: Clique aqui
Clique aqui (Nova guia)

Hyperlinks Locais

São hyperlinks hospedados em um mesmo servidor. Exemplo: Hyperlink Local

Hyperlinks em Imagens

É possível deixar uma imagem clicável, direcionando para uma outra referência.

Hyperlinks Internos

São hyperlinks que referenciam dentro itens do mesmo documento. Exemplo:
