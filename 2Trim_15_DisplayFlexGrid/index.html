Display

Display é uma propriedade do CSS que permite ao desenvolvedor, manipular a visualização da página que o usuário irá usar.br
Geralmente é utilizado para poder se ter uma organização limpa e ordenada dos objetos (tags como img, h1, p, div e etc)

Flex

O valor "Flex" para a propriedade Display, permite uma alta manipulação dos objetos dentro de um item-pai, é atualmente a versão mais nova.
