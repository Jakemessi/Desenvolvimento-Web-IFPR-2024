O Projeto

O projeto Trilha de Aprendizado consistiu numa simples porém importantíssima atividade orientada pelo Professor Francis Baranovski
Esta atividade resume em organizar todos os conteúdos aprendidos de forma que seja através de uma barra de navegação, seja possível acessá-las.

Desta forma, o aluno, caso faça essa atividade por completo com o mínimo de perfeição, ou seja, de qualidade suficiente, provará que o aluno está apto a criar sites básicos mesmo que até o momento não tenha visto sobre JavaScript
