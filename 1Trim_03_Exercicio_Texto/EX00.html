HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo, e que podem ser vinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

Ou para deletar uma parte... Como roubar um banco

Tag ins

A tag ins basicamente sublinha o contexto texto
Usada para marcar textos adicionados após uma revisão/atualização
Ou seja, útil para changelog/notas de atualização

Tag sub

Essa tag sub serve para deixar letras no meio da linha
Útil para escrever fórmulas químicas como H2O

Olha o terremoto capa

C17H21NO4C é a fórmula da benzoilmetilecgonina

Tag sup

A tag sup serve para colocar letras acima de seu nível, como se fossem expoentes

52 equivale a 25

O 1o cavalo a passar a linha de chegada foi do vaqueiro Geovane

O 39o Presidente do Brasil é o Luiz Inácio Lula da Silva
